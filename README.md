# Presentación

El presente proyecto final tiene como objetivo investigar y presentar información relacionada con la carrera de Diseño y Programación Web, destacando sus principales áreas, herramientas fundamentales, perfil profesional y campo laboral dentro del mundo tecnológico.

A través de este trabajo se busca mostrar la importancia de la combinación entre el diseño creativo y la programación, elementos esenciales para la creación de sitios web y soluciones digitales modernas.

Asimismo, este proyecto permite aplicar el trabajo colaborativo mediante el uso de Git y GitHub, herramientas que facilitan la organización, control y seguimiento de los cambios realizados durante el desarrollo del proyecto.
